MMMPseudoRandomSequence

Objective-C

@interface MMMPseudoRandomSequence : NSObject

Swift

class MMMPseudoRandomSequence : NSObject

A wrapper for a sequence of randomly looking integers that is convenient to use for testing (so every sequence of pseudo-random events can be reproduced many times). For non-test purposes use arc4random() and friends instead.

A seed can be set for the standard rand() function as well, but its usage cannot be restricted to a single entity, so a sequence can be disrupted.
